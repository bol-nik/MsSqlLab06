Лаб. раб. №6

1. Вывести идентификаторы и фамилии тех клиентов, у которых есть один или несколько заказов.
2. Вывести идентификаторы, имена и фамилии сотрудников, продававших товар.
3. Вывести идентификаторы, имена и фамилии сотрудников, которые ничего не продавали.
4. Поместить в одну результирующую таблицу два столбца: название таблицы и количество записей для таблиц: SALE, PRODUCT, SPERSON. Выходные данные отсортировать в возрастающем порядке значений второго столбца.

В работе использовать составные запросы.

ПРИМЕРЫ:
1.	Вывести фамилии клиентов и имена и фамилии сотрудников в один столбец:
select c_name spisok from customer
union all
select sp_name from sperson;

2.	Вывести id сотрудников, которые продавали товар:
select sp_id from sperson
intersect
select sp_id from sale
order by 1;

3.	Вывести имена и фамилии сотрудников, которые продавали товар:
select sp_name from sperson 
where sp_id in
(select sp_id from sperson
intersect
select sp_id from sale)
order by 1;

4.	Вывести id сотрудников, которые ничего не продавали:
select sp_id from sperson
except
select sp_id from sale
order by 1;
